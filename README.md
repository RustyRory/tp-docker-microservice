# TP Docker et Kubernetes - MDS Architecture

Ce projet contient 5 TP réalisés pour apprendre la conteneurisation et l'orchestration d'applications :

- Docker
- Docker Compose
- Kubernetes.
- Chaque TP propose un exemple pratique de microservices ou d'application multi-services.

Les PDFs originaux fournis par le professeur sont également inclus.
